Calendriers et contacts

Cette page vous aidera à accéder aux calendriers et aux contacts depuis n'importe quel client, mobile ou ordinateur, ou depuis un interface Web.

Interface Web

Une fois installé, vous pouvez accéder aux calendriers et aux contacts à l'aide de SOGo une interface Web réactive, à l’adresse https://sogo.[network.domain]/.

Entre autres choses, vous pouvez accéder aux e-mails, aux contacts, aux calendriers et aux tâches dans un navigateur Web, à partir d'un un ordinateur de bureau ou un appareil mobile.

Détails

  • Rédigez et envoyez des e-mails.
  • Accédez à vos agendas, partagez-les ou déléguez-les avec d'autres membres, ou même abonnez-vous à des agendas en ligne.
  • Créez ou modifiez des événements dans votre liste d'agendas.
  • Accédez à vos carnets d'adresses, partagez-les ou déléguez-les avec d'autres membres.
  • Configurez vos paramètres de messagerie, comme les règles de vacances, de transfert et de filtrage.
  • Modifiez votre mot de passe.

Plus de détails sur le site SOGo, la solution utilisée : https://www.sogo.nu/about.html#/features

Linux

Les clients Linux les plus populaires sont compatibles avec les standards CALDAV / CARDDAV.

  • Mozilla Thunderbird
  • Évolution de Gnome

Voir les instructions sur la page creation de compte.

Microsoft Windows

Pour Microsoft Windows, il est recommandé d’utiliser Mozilla Thunderbird, qui prend en charge nativement les normes CALDAV / CARDDAV.
Voir les instructions sur la page creation de compte.

eM Client est un client de messagerie pour Windows et macOS, qui prend en charge nativement les standards ci-dessus.

Pour Microsoft Outlook, vous pouvez utiliser le caldav synchroniser ouvert plugin source.

Appareils Apple

Les appareils Apple prennent en charge nativement les normes. De nombreux tutoriels existent sur internet, et sont renvoyés via une recherche sur le web.

Application DAVx sur Android

DAVx est un logiciel tout-en-un CalDAV/CardDAV/WebDAV gratuit et open source. solution, synchronisant les contacts et les calendriers avec votre appareil Android et le serveur. Une fois installées et configurées, les applications Android natives auront un accès transparent au les contacts et les calendriers. Vous pouvez le télécharger depuis Google Play. Une fois installé, suivez les étapes suivantes pour importer vos calendriers.

  1. Ajouter un compte sur DAVx
  2. Le logiciel DAVx détecte les paramètres
  3. Définir le nom du compte
  4. Voir la liste des carnets d’adresses
  5. Voir la liste des calendriers
  6. Abonnez-vous aux calendriers Web (facultatif)
