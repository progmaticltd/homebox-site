Calendars and contacts

This page will help you to access calendars and contacts from any client, mobile or desktop, or from a web interface.

Web interface

Once installed, you can access calendars and contacts using SOGo a responsive web interface, at the address https://sogo.[network.domain]/.

Amongst other things, you can access emails, contacts, calendars and tasks in a web browser, from a desktop or a mobile device.

Details

  • Compose and send emails.
  • Access your calendars, share or delegate them with other members, or even subscribe to online calendars.
  • Create or edit events in your calendars list.
  • Access your address books, share or delegate them with other members.
  • Configure your emails settings, like vacation, forward and filter rules.
  • Change your password.

More details on the SOGo site, the solution used: https://www.sogo.nu/about.html#/features

Linux

The most popular Linux clients are compatible with the CALDAV / CARDDAV standards.

  • Mozilla Thunderbird
  • Gnome Evolution

See the instructions on the account set-up page.

Microsoft windows

For Microsoft Windows, it is recommended to use Mozilla Thunderbird, that natively supports the CALDAV / CARDDAV standards.
See the instructions on the account set-up page.

eM Client is an email client for Windows and macOS, that natively supports the standards above.

For Microsoft Outlook, you can use caldav synchronizer open source plugin.

Apple devices

Apple devices natively supports the standards. Many tutorials exists on internet, and are returned using a search engine.

Mobile DAVx client on Android

DAVx is an all-in-one CalDAV/CardDAV/WebDAV free and open source solution, synchroning contacts and calendars with your android device and the server. Once installed and configured, the native android applications will have transparent access to the contacts and the calendars. You can download it from Google play. Once installed, follow the following steps to import your calendars.

  1. Add an account on DAVx
  2. The DAVx software detect the settings
  3. Set the account name
  4. See the address books list
  5. See the calendars list
  6. Subscribe to web calendars (optional)
