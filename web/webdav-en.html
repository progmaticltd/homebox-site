Personal file servers

This page will describe how to use the webdav server.

WebDAV server

Once installed, you can access your WebDAV server using any compatible client, at the address https://webdav.[network.domain]/.

Linux

Linux desktop environments are coming with compatible WebDAV clients, for instance, Nautilus.

Mobile DAVx client on Android

DAVx is an excellent free and open source WebDAV client. Once installed and configured, the native android applications will have transparent access to the storage space. You can download it from Google play. Once installed, follow the following steps to configure your storage space.

  1. Add an account on DAVx
  2. The DAVx software detect the settings
  3. Set the account name
  4. Mount the WebDAV folder
  5. List of mounted webdav folders
