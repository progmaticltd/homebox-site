Jabber server

A Jabber server is installed and configured, and can be used with any XMPP client. The following features are supported and configured

  • Strong encryption settings.
  • Audio and Video calls support.
  • Automatic server detection with modern clients using service DNS records.
  • Client to server, and server to server communications.
  • File sharing and image previews.
