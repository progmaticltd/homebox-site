Serveur git personnel

Vous pouvez installer un serveur git personnel pour chaque utilisateur, principalement utile pour les projets personnels ou les fichiers initialisation.

Serveur serveur Git

  • Les dépôts sont accessibles via SSH, pas de front-end web.
  • Les dépôts Git sont automatiquement créés lors du premier push.
  • Validation des noms de dépôts
  • Les dépôts sont stockés dans /home/archives par défaut.

Exemple de configuration

                
                    users:
                      - uid: alice
                        [...]
                        ssh:
                          - comment: git-custom-key
                            type: ssh-ed25519
                            data: AAAAC3NzaC1lZDI1NTE5AAAAUmspI/sxaHxkpPx0KCunUwhtWdX8ERWwyl/CHfPVxDYu
                            usage: git
                      - uid: bob
                        [...]
                        ssh:
                          - comment: git-custom-key
                            type: ssh-ed25519
                            data: AAAAC3NzaC1lrVT63F9QZDe3O9iqLYI1swRgGAcEHBo2zIhj42iisEbzQXxbpwEufmbS
                            usage: git

Utilisation

                
                    git remote add perso alice@domain.com:repo-name.git
                    git push --all --tags perso
