Personal git server

You can install a personal git server for each user, mostly useful for personal projects or dot files.

Git server server

  • Repositories are accessed over SSH only, no web front-end.
  • Git repositories are automatically created on first push.
  • Repository names validation.
  • Repositories are stored in /home/archives by default.
  • Directory and files are only readable by the user.

Configuration example

                
                    users:
                      - uid: alice
                        [...]
                        ssh:
                          - comment: git-custom-key
                            type: ssh-ed25519
                            data: AAAAC3NzaC1lZDI1NTE5AAAAUmspI/sxaHxkpPx0KCunUwhtWdX8ERWwyl/CHfPVxDYu
                            usage: git
                      - uid: bob
                        [...]
                        ssh:
                          - comment: git-custom-key
                            type: ssh-ed25519
                            data: AAAAC3NzaC1lrVT63F9QZDe3O9iqLYI1swRgGAcEHBo2zIhj42iisEbzQXxbpwEufmbS
                            usage: git

Usage

                
                    git remote add personal alice@domain.com:repo-name.git
                    git push --all --tags personal
